1- Conceitue os termos "dado" e "informação" no que se refere ao seu emprego em processamento de dados.

	Dado é o menor nível de informação, representam uma unidade de informação que pode ser processada por um identificador.

	Informação é a reunião dos dados que são capazes de realizar um resultado.

2- Caracterize as etapas pricipais de um processamento de dados.

	Os dados são coletados através um hardware, gravados e enviados ao processador e então recebidos por um software que irá analizá-los.

3- Conceitue um sistema. Cite dois exemplos práticos de organizações sistêmicas na vida real.

	Sistema é um conjunto de elementos com alguma interação regular criados para realizar um objetivo definido. Ex: Sistema financeiro, sistema ferroviário.

4- O que você entende por um programa de computador?

	Algo feito totalmente dentro de um computador a partir de linguagens de programação, ou seja, totalmente no software.

5- Conceitue os termos hardware e software.

	Hardware é conjunto físico de componentes e o software é o sistema digitalizado, que só funciona por conta do hardware.

6- O que é e pra que serve uma linguagem de programação? Cite exemplos.

	É um conjunto de regras usadas para criar programas no software (dentro do computador e não externamente).

7- Cite três componentes eletromecânicos de um computador básico.

	Fan, cooler e e não sei mais algum.

8- Cite três componentes eletrônicos de um computador básico.

	Cpu, Gpu e Ram.

9- Qual a diferença entre um computador e um sistema eletrônico?

	A diferença é que o computador recebe e processa dados enquanto um sistema eletrônico só faz funções específicas e limitadas.

10- Conceitue a Lei de Moore e justifique se ela se aplica a tecnologia atual.

	A Lei de Moore previu que o número de transístores em um circuito integrado dobraria anualmente. 

	Segundo Carl Anderson a Lei de Moore não funciona mais, pois os engenheiros já desenvolveram sistemas que não exigem mais tantos recursos do processador.
